# 点滴音乐 

### 微信小程序 音乐播放器

后台使用了网易云音乐Node.js API  https://binaryify.github.io/NeteaseCloudMusicApi/#/ 

### 完成内容

#### 1.首页

展示了搜索框、轮播图、四个快捷入口、最新音乐、精选歌单、推荐MV、精选电台、最新专辑等。
